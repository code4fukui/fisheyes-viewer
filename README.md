# fisheyes-viewer

> 日本語のREADMEはこちらです: [README.ja.md](README.ja.md)

Viewer for spatial photos and videos captured with Canon Dual Fisheye cameras, for use on VR/AR headsets like Quest and Vision Pro.

## Demo

- [Slideshow of 8K spatial photos from OkakiNEO at Mouemon farm in Sabae City](https://code4fukui.github.io/fisheyes-viewer/slideshow?url=https://tf0.code4fukui.org/vr180/2024-05-14-mouemon/mouemon.m3u8)
- [4K M3U8 video from OkakiNEO at Mouemon farm in Sabae City](https://code4fukui.github.io/fisheyes-viewer/?url=https://tf0.code4fukui.org/vr180/2024-05-14-mouemon/kome.m3u8)

## Features

- Displays spatial photos and videos captured with Canon Dual Fisheye cameras
- Supports playback on VR/AR headsets like Oculus Quest and Apple Vision Pro
- Uses Three.js for rendering
- Includes a slideshow mode with automatic transitions

## Requirements

- A modern web browser with WebXR support

## Usage

1. Open the [fisheyes-viewer](https://code4fukui.github.io/fisheyes-viewer/) page in your browser.
2. If you have a VR/AR headset, click the "Start AR" or "Start VR" button to enter immersive mode.
3. Alternatively, you can view the content on a regular screen by using the mouse or touch controls to navigate.
